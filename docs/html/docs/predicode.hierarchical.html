predicode.hierarchical.weight_init.weight_init(char, **kwargs)¶

Initialize weights according to method.

Initialize weights according to the specified method and the additional input arguments as a shortcut for pc.weight_init_*(...)(...).

If passed a numpy array or a function, it returns the object.

Parameters

  • input_dimensions – Number of input dimensions.

  • latent_dimensions – Number of latent dimensions. Must be less than the

input dimensions. :param input_data: Input data as a two dimensional numpy array. One sample per row and one dimension per column. :param char: Which method?

Returns

Two dimensional numpy array with one rotation per row.
