Interface to the minimal model, linear predictive coding with a single hidden layer.

class predicode.hierarchical.interfaces.minimal_model.MinimalHierarchicalModel(input_data, weights='pca', latent_dimensions=None, use_bias=False, state_learning_rate=1, **kwargs)¶

Bases: object

Interface to the minimal hierarchical model.

The minimal hierarchical model is given by a linear predictive coding model with one densely connected, linear layer.

Parameters

input_data – A numpy array, where the columns correspond to the

dimensions and the rows correspond to the number of observations. :param input_dimensions: Input dimensions. :param n_observations: Number of observations. :param latent_dimensions: Dimensions of the latent layer. :param weights: An object that can be fed through :fun:~predicode.weight_init. :param use_bias: Should the model use an additional bias layer?
